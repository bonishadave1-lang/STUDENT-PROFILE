# 🎓 ATMIYA Student Portal - Pro Edition 2026

A modern, responsive **Student Management System** developed for academic record tracking. This portal demonstrates a Single Page Application (SPA) architecture with advanced features like dynamic data simulation and automated report generation.


---

## ✨ Key Features
* **Dynamic Data Generation:** Automatically simulates 400+ student records with randomized academic scores and attendance.
* **Real-Time Analytics:** Dashboard cards show total student strength and real-time failure counts.
* **Search & Filter:** Instant search by Student Name, Roll Number, or Result Status (Pass/Fail).
* **PDF Transcript Export:** * Generate a full Master List PDF.
    * Print individual student academic profiles/marksheets.
* **Responsive UI:** Glassmorphism design built with Bootstrap 5, optimized for mobile and desktop.

## 🛠️ Technical Stack
* **Framework:** AngularJS 1.8 (Directives, Services, and ngRoute)
* **Styling:** Bootstrap 5 & Custom CSS3 (Glassmorphism effect)
* **Logic:** JavaScript (ES6)
* **Icons/Fonts:** Emoji-based iconography and Segoe UI typography

## 📂 Project Structure
* `index.html`: The core Single Page Application.
* `GlobalController`: Manages student data across the app.
* `DataStore Service`: Logic for generating and storing randomized student records.
* `Print Media Queries`: Specialized CSS for professional PDF exports.
